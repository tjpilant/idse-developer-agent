# Handoff Templates README

> üìÇ **Directory:** `idse-governance/templates/handoff_templates/`
>
> This directory contains all **handoff and feedback templates** used by the automated governance system that manages dual-LLM collaboration (Claude ‚Üî Codex) under the **Intent-Driven Systems Engineering (IDSE)** framework.

---

## üß≠ Purpose

These templates define the **structure of communication artifacts** generated during each governance cycle:

* Ensuring all transitions between Claude and Codex are documented
* Maintaining a full audit trail of IDSE stage transitions
* Linking each exchange to the active stage, plan, and constitutional reference

Each template is used automatically by the `.cursor/tasks/governance.py` script or manually by IDE tasks in VS Code.

---

## üìÑ Templates Overview

| Template                           | Purpose                                                                                 | Generated By    | Output Destination                                               |
| ---------------------------------- | --------------------------------------------------------------------------------------- | --------------- | ---------------------------------------------------------------- |
| **`claude_to_codex_template.md`**  | Used when Claude completes a build or planning phase and hands off to Codex for review. | Claude          | `idse-governance/feedback/handoff_<cycle_id>_claude_to_codex.md` |
| **`codex_to_claude_template.md`**  | Used when Codex finishes review and returns control to Claude for refinement.           | Codex           | `idse-governance/feedback/handoff_<cycle_id>_codex_to_claude.md` |
| **`handoff_feedback_template.md`** | Used after a full Claude ‚Üî Codex cycle to summarize learnings, metrics, and feedback.   | System / Claude | `idse-governance/feedback/handoff_summary_<cycle_id>.md`         |
