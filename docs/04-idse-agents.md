# IDSE Agent Framework

IDSE uses a multi-agent conceptual model:

## Intent Agent
Extracts purpose, outcomes, constraints.

## Context Agent
Discovers scale, stack, compliance, risks, environment.

## Specification Agent
Produces structured, testable system behavior.

## Plan Agent
Creates architectural plan mapping spec → design → components.

## Task Agent
Breaks plan into atomic tasks.

## Implementation Agent
Generates code + tests to satisfy tasks.

All Agents follow the IDSE Constitution.
